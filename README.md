# 获取中国国家统计局网站数据

数据源：http://www.stats.gov.cn/

🚨 注意：默认分支现在是 `main`，而不是 `master`（2025 年 5 月 24 日修改）。

## PyPi安装

```bash
pip install cn-stats
```

## cn-stats使用

例：查询宏观数据

```python
from cnstats.stats import stats

result = stats(zbcode='A010101', datestr='202201')

print(result)
```

例：查询城市数据

```python
from cnstats.stats import stats

result = stats(zbcode='A010101', datestr='202201', regcode='370200', dbcode='csyd')

print(result)
```

## 命令行中使用

使用帮助

```bash
python -m cnstats --help
usage: cn-stats [-h] [--tree] [--list-regcode] [--dbcode [DBCODE]] [--regcode [REGCODE]] [zbcode] [date]

获取中国国家统计局网站数据Python包

positional arguments:
  zbcode               指标代码
  date                 查询日期

options:
  -h, --help           show this help message and exit
  --tree               列出指标代码
  --list-regcode       列出地区代码, 默认列出分省地区代码，列出主要城市代码添加 --dbcode csnd 参数
  --dbcode [DBCODE]    数据库代码, 默认: hgyd(宏观月度), 可选: hgjd(宏观季度), hgnd(宏观年度), fsyd(分省月度), fsjd(分省季度), fsnd(分省年度), csyd(城市月度), csjd(城市季度), csnd(城市年度)
  --regcode [REGCODE]  地区代码, 例如: 110000 是北京市
```

例：获取2022年1月货币供应量数据

```bash
$ python -m cnstats A0D01 202201

A0D0105 202201 106188.87

```

```bash
python -m cnstats A01010101 202112,202201 --regcode 110000
```
